# Legal Script Kit — 律师口播文案创作工具

做律师短视频，最难的不是法律功底，而是**把复杂的案子变成观众听得懂、愿意听的话**。

这个工具就是帮您解决这个问题的。

## 它解决了什么问题

很多律师做短视频面临三个困境：

- **没时间写** — 案子本身就忙，哪有空逐字推敲文案
- **写出来没人看** — 专业归专业，但开场 3 秒留不住人
- **坚持不下去了** — 发了十几条没反馈，不知道问题出在哪

这个工具把写口播文案这件事，从「凭感觉写」变成了「按方法论出」。

## 它能做什么

| 您给它 | 它给您 |
|--------|--------|
| 一个判决书或案情摘要 | 可以直接对着镜头念的口播文案 |
| 一个法律观点 | 一篇有钩子、有案例、有态度的普法文案 |
| 一个奇葩案件 | 一篇能吃瓜、有反转、能出圈的爆款文案 |

## 内置的方法论

不是随便生成一段文字，而是按您选择的套路来写：

**5 种文案类型**
- 分享型 — 讲案子像讲故事，适合日常引流
- 分析型 — 拆法律逻辑，适合立专业人设
- 吃瓜型 — 反转+奇葩，适合出爆款
- 观点型 — 反常识开场，适合树立权威
- 观点干货版 — 纯知识输出，不做案件搬运工

**8 种爆款钩子**
从「建议祈使」到「情绪钩子」，每种开场方式都给出了案例句，您选一个最贴案子气质的就行。

**3 种人设风格**
犀利毒舌、亲切邻家、幽默吐槽 — 您是什么风格，文案就按什么语气来。

## 谁在用

面向一线执业律师，特别是：
- 刚开始做短视频、不知道怎么下手的新人
- 已经有账号但内容输出效率低的律师
- 想系统化生产内容、不想每次从零开始写的团队
