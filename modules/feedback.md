# 模块：用户反馈

## 何时执行

用户对输出结果提出修改意见时。

## 反馈类型

| 类型 | 处理方式 |
|------|---------|
| 文案类型调整 | 重新进入 workflow/02-script-type.md 选择新类型 |
| 人设风格调整 | 切换 rules/persona-styles.md 对应风格 |
| 钩子调整 | 参考 workflow/03-hook-routing.md 重选钩子 |
| 内容修改 | 局部调整对应段落 |
| 平台调整 | 按 rules/platform-adaptation.md 适配 |

## 反馈记录

每次修改后，简要记录：
- 用户修改了什么
- 为什么修改
- （可选）将修改后的模板存入本地
