# 口播文案撰写模板库

> 本文件包含：文案类型撰写规范 / 平台适配细则 / 案件类型叙事模板 / 完整示例

---

## 一、文案类型撰写规范

### 1. 分享型 — 故事化案件分享

**定位：** 悬念开场、情感共鸣，适合短视频引流。
**结构：** '他来找我的时候' → 事实（从当事人角度）→ 我们的判断 → 我们做了什么 → 结果宣告 → 对观众说

**每段规范：**

| 段落 | 内容 | 视角 |
|------|------|------|
| 立场开场 | 钩子句 + 律师立场表态 | 一句话说清价值观 |
| 案件概览 | '当事人来找我的时候' + 当事人状态 + 发生了什么 | 从当事人第一视角 |
| 法律分析一 | '我们一看就知道' + 核心法律判断 | 用"我们" |
| 法律分析二 | '我们帮他' + 做了什么（取证/起诉/谈判） | 用"我们" |
| 结果宣告 | '最终' + 判决/调解结果 + 当事人的反应 | 替当事人表达 |
| 对观众说 | 关键建议 + 互动/转化引导 | 直接对镜头 |

---

### 2. 分析型 — 深度案件分析

**定位：** 逻辑拆解、争议焦点，适合专业普法。
**结构：** 问题概览 → 争议焦点拆解 → 法理分析 → 判决逻辑 → 启示

**每段规范：**

| 段落 | 内容 | 视角 |
|------|------|------|
| 问题概览 | '今天讲一个很多人会遇到的案子' + 案件基本事实 | 中立但带立场 |
| 争议焦点一 | '这个案子第一个争议焦点是' + 法律分析 | 专业但易懂 |
| 争议焦点二 | '第二个关键是' + 进一步分析 | 专业但易懂 |
| 法理分析 | '法律上怎么看的' + 法条/原理 | 用"我们"引出 |
| 判决逻辑 | '法院为什么这么判' + 判决逻辑拆解 | 解读但不中立 |
| 启示 | '所以你要记住' + 对观众的实用建议 | 直接对镜头 |

---

### 3. 吃瓜型 — 奇葩/娱乐化案件

**定位：** 吃瓜式开场、层层反转，适合引流爆款。
**结构：** 奇葩钩子 → 当事人出场 → 离谱展开 → 我们出手 → 反转 → 感叹

**每段规范：**

| 段落 | 内容 | 提示 |
|------|------|------|
| 奇葩钩子 | 一句话让人好奇 | "我做了十年律师，第一次遇到这种事" |
| 当事人出场 | '当事人来找我的时候' + 当事人的离谱处境 | 带幽默感 |
| 离谱展开 | 对方做了什么奇葩操作 | 带吐槽感 |
| 我们出手 | '我们一看这情况' + 我们的应对 | 用"我们" |
| 反转来了 | 结果出人意料 | 制造"啊？"效果 |
| 感叹 | '说实话我看了都' + 感叹+对观众说 | 轻松收尾 |

---

### 4. 观点型 — 法律观点输出

**定位：** 反常识观点开场、观点先行，案子只是佐证。
**结构：** 反常识观点 → 法律逻辑 → 案子佐证（一笔带过）→ 行动建议

**关键原则：**
- 消费者的心理是"我想知道一个我不知道的事"——不是听故事
- 案子只是论据，不超过 2 句话
- 结尾必须是可操作的行动建议

**每段规范：**

| 段落 | 内容 | 注意 |
|------|------|------|
| 反常识观点开场 | 钩子句 + 观点本身 | 跟具体案子无关 |
| 法律逻辑 | '为什么？法律上是这样看的' | 核心价值所在 |
| 案子佐证 | '比如我们办过一个案子' + 一笔带过 | **不超过 2 句话** |
| 行动建议 | '所以记住' + 可操作的建议 | 直接对镜头 |

## 二、平台适配细则

| 要素 | 抖音（默认） | 视频号 | 小红书 |
|------|-------------|--------|--------|
| 开场 | 前 3 秒即钩子，节奏极快 | 可铺垫 5-8 秒，理中客 | 情感共鸣优先，标题辅助 |
| 语速 | 快（4.5-5字/秒） | 正常（3.5-4字/秒） | 正常偏慢（3-3.5字/秒） |
| 情绪 | 夸张、戏剧化 | 温和、理性 | 温暖、共情 |
| 互动 | "评论区说说你的看法" | "您觉得呢？欢迎讨论" | "姐妹们遇到过吗" |
| 标题 | 情绪化、悬念型 | 事实型、知识型 | 笔记式、干货型 |
| 长度 | 30-60秒（180-360字） | 60-90秒（210-360字） | 15-30秒（90-180字） |

---

### 5. 观点干货版 — 纯干货，无案情

**定位：** 不依赖具体案件，纯法律知识输出。核心不靠模板套话，靠流量、信任、人设三大底层逻辑持续出爆款。
**适用题材：** 婚姻、债务、劳动、房产、刑事等所有法律领域。
**结构：** 无固定结构，六大创作心法自由组合（详见 SKILL.md §4.7）。

**核心区别：**

| 维度 | 观点型 | 观点干货版 |
|------|--------|-----------|
| 驱动力 | 反常识观点 + 案子佐证 | 大众认知误区 -> 实务真相 |
| 结构 | 固定四段 | 无固定，心法自由组合 |
| 案子 | 必须有，一笔带过 | 不需要，纯知识 |
| 人设 | 规则的翻译者 | 客观理性的避坑向导 |
| 适用 | 有典型案件时 | 任何法律题材 |

**每段规范（仅供参考，非强制）：**

| 段落 | 内容 | 核心要点 |
|------|------|---------|
| 开场 | 全民误区 -> 认知反差 | 前3秒推翻大众深信不疑的认知 |
| 代入 | 还原普通人维权误区场景 | 让用户看到"这说的就是我" |
| 专业 | 官方口径 + 裁判规则 | 用权威堵质疑，不靠个人经验 |
| 落地 | 本地实操 + 差异化细节 | 通用法条人人可查，本地才是稀缺 |
| 价值 | 量化利弊 + 算账避坑 | 帮用户省钱省时间，不怂恿打官司 |
| 人设 | 反营销感收尾 | 劝阻盲目维权，建长期信任 |

## 三、案件类型叙事模板（分享型适用）

以下模板按案件类型给出**分享型**的叙事框架，其他文案类型也可参考其中的代理视角细节。

# 案件叙事结构模板

## 案件类型与推荐叙事结构

### 1. 交通事故/人身伤害赔偿案
> **推荐钩子：** C反常识对比（"全责≠全赔"） / F数字锚点（金额对比）
> **推荐开场钩子句：** "对方全责，保险公司只赔2万——我当时就跟当事人说，这钱我们不能认。"

**结构：'他来找我的时候'→ 事实（从当事人角度）→ 我们的判断 → 我们做了什么 → 结果宣告 → 对观众说**

示例框架：
- **当事人出场**："当事人来找我的时候，[受伤细节/状态]。"
- **对方态度**："对方[保险公司/个人]上来说[态度]——我当时就告诉他这个不能接受。"
- **我们做了什么**："我们帮他做了[取证/起诉/谈判]，关键抓住了[法律点]。"
- **我们争的是什么**："在法庭上/谈判桌上，我们跟对方争的核心就一个——"
- **结果宣告**："法院最终判了[金额]——我们跟当事人说过的话，一分不少。"
- **对观众说**："如果[类似情况]发生在你身上，记住[关键建议]——别一个人扛，有人能帮你。"
- **结尾**（引流型）："遇到过类似情况吗？评论区说说你的经历。"
- **结尾**（转化型）："我是李律师，有交通事故问题，私信我。"

### 2. 婚姻家事/离婚纠纷
> **推荐钩子：** G情绪钩子（情感共鸣） / A建议祈使（"婚前必知"类）
> **推荐开场钩子句：** "当事人来找我的时候，整个人瘦了一圈——结婚[X]年，走到这一步，最让人心寒的不是分开，而是..."

**结构：'她来找我的时候'→ 她经历了什么 → 我们怎么帮她 → 对方什么态度 → 结果 → 替她说的最后几句话**

示例框架：
- **当事人出场**："当事人来找我的时候，[状态描述——比如'整个人瘦了一圈''眼圈红了又忍住']。"
- **她的故事**："结婚[X]年，[曾经的美好]，后来[矛盾升级]..."
- **我们的判断**："她问我怎么办，我说[关键判断]。法律上这确实[法律点]，但我们能帮她争取的是[具体目标]。"
- **对方态度**："对方[对方的行为——比如'一分钱都不肯给''连孩子都不让见']，说实话我看了都..."
- **结果**："最终[结果——抚养权/财产分割/赔偿]，当事人说[她的反应]。"
- **替她说的话**："我想替当事人说一句——婚姻法保护的是财产，不是感情。但至少，你的底线有人帮你守住。"
- **结尾**（引流型）："你觉得当事人做得对吗？评论区说说你的看法。"
- **结尾**（转化型）："我是李律师，用法律帮你守住底线。"

### 3. 刑事辩护
> **推荐钩子：** C反常识对比（"你以为这就构成犯罪了？"） / B顿悟发现（"我们进去一看就发现了问题"）
> **推荐开场钩子句：** "当事人被带走的那天，家属找到我——我看了材料第一反应是：这个指控站不住。"

**结构：'家属找到我的时候'→ 他面对什么 → 我们发现了什么 → 我们在法庭上说了什么 → 结果 → 我想说的**

示例框架：
- **家属出场**："当事人被[拘留/逮捕]的第[X]天，家属找到我。他/她当时[状态——比方'整个人是崩溃的']。"
- **面对什么**："指控是[罪名]，如果成立，面临[刑期]。但我在阅卷的时候发现了一个关键问题——[辩点]。"
- **我们的策略**："我们在法庭上争的是：[辩护要点]。我认为[法律上的核心论点]。"
- **结果**："最终——从[原指控/刑期]到[最终结果]。当事人出来的时候，[状态/他说的话]。"
- **替他说的话**："我想说的是，[法律道理]。但这个案子让我更想说的是——[从人性角度的感慨]。"
- **结尾**："我是李律师，如果你或家人遇到刑事问题，越早找律师越主动。"

### 4. 合同/经济纠纷
> **推荐钩子：** D如何方法（"签合同注意这3点"） / A建议祈使（"千万不要这样签"）
> **推荐开场钩子句：** "当事人来找我的时候，拿着一份合同——他签了个字，赔了[X]万。"

**结构：'他拿着合同来找我的时候'→ 合同里有什么陷阱 → 我们怎么帮他的 → 结果 → 对你说**

示例框架：
- **当事人出场**："当事人来找我的时候，拿着一份[合同类型]，一脸愁容——他刚发现[问题]。"
- **我们一看**："我翻了一遍合同，一眼就发现了问题——[关键陷阱条款/违约点]。说实话，这种条款我见得太多了。"
- **我们做了什么**："我们帮他[诉讼/仲裁/谈判]，在这个点上我们跟对方争的是[核心争议]。"
- **结果**："最终[结果]，当事人说[他/她的反应]。"
- **对你说**："合同里的[某条款]一定要看清楚，不然[后果]。签之前拿不准的，问我一句，省得后面吃大亏。"
- **结尾**："我是李律师，合同问题拿不准的，私信我。"

### 5. 劳动工伤/劳动争议
> **推荐钩子：** E身份圈定（"打工人必知"） / D如何方法（维权方法）
> **推荐开场钩子句：** "当事人来找我的时候，说在公司干了[X]年，被辞退一分补偿都没有——我一听就知道这不合法。"

**结构：'他来求我的时候'→ 公司对他做了什么 → 法律上这算什么 → 我们怎么帮他 → 结果 → 对打工人说**

示例框架：
- **当事人出场**："当事人来找我的时候，[状态——比方'刚被辞退，一脸茫然']。他在[公司]干了[X]年，做[岗位]。"
- **公司做了什么**："公司[违法事实]（不签合同/不交社保/违法辞退/不认定工伤）。我当时听了就觉得——这公司太欺负人了。"
- **我们的判断**："法律上，这属于[法律定性]。我们告诉他，这个我们能帮你争。"
- **我们做了什么**："我们帮他[仲裁/诉讼/投诉]，关键点是[法律依据]。"
- **结果**："最终拿到了[赔偿/认定/补偿]——他跟你说[他的反应]。"
- **替他说的话**："我想替他说一句——[法律条款]保护的就是你这样的劳动者。公司不是你的家，但法律是你的盾。"
- **结尾**："你在公司遇到过这种事吗？评论区说说。"

### 6. 遗产继承纠纷
> **推荐钩子：** G情绪钩子（"老人走了，留下的是一地鸡毛"） / A建议祈使（"提前写好遗嘱"）
> **推荐开场钩子句：** "当事人来找我的时候说，父亲刚走，兄弟姐妹已经翻了——老人留下的不是遗产，是一地鸡毛。"

**结构：'她来找我的时候'→ 老人走了以后发生了什么 → 法律上谁对谁错 → 我们怎么调解/诉讼 → 结果 → 替她说的**

示例框架：
- **当事人出场**："当事人来找我的时候，[状态描述]。老人刚走，兄弟姐妹已经因为[房产/存款]翻脸了。"
- **矛盾**："[被继承人]有[遗产情况]，子女[几人]。因为[遗嘱缺失/分配不公/抚养事实]，当事人被[对方做了什么]。"
- **我们的判断**："法律上，[法律分析]。我们告诉当事人：你有权利[具体主张]。"
- **我们做了什么**："我们通过[调解/诉讼/遗嘱鉴定]，最终[分配方案]。在这个过程中，当事人说[她的感受]。"
- **替她说的**："遗产案赢的从来不是钱，是安心。但我想替当事人说的是——有些底线，是法律帮你守住的。"
- **结尾**："家家有本难念的经。提前写好遗嘱，是对家人最大的温柔。"

### 7. 知识产权/侵权
> **推荐钩子：** G情绪钩子（"心血被偷"） / D如何方法（"被侵权了怎么办"）
> **推荐开场钩子句：** "当事人来找我的时候，气得手都在抖——他辛辛苦苦做的[作品]，别人拿来就用，一分钱没给。"

**结构：'他来找我的时候'→ 他花了多少心血 → 对方怎么偷的 → 我们怎么维权 → 结果 → 替创作者说**

示例框架：
- **当事人出场**："当事人来找我的时候，[状态描述——比方'气得手都在抖']。他花了[时间/金钱]做了[作品/品牌/发明]。"
- **发现侵权**："突然发现[对方]在[平台/市场]上直接拿来用——连改都没改。我当时看了，第一反应是：这也太明目张胆了。"
- **我们做了什么**："我们帮他[取证/公证/诉讼]。过程[艰难/顺利]，但我们坚持的是[法律依据]。"
- **结果**："最终[赔偿/停止侵权/道歉]。当事人说[他的反应]。"
- **替他说的话**："我想说的不是法条，而是——知识产权不是大公司的专利。你的创意，法律帮你保护。"
- **结尾**："你的作品被侵权过吗？评论区聊聊。"

### 8. 房产纠纷
> **推荐钩子：** F数字锚点（金额锚点） / A建议祈使（避坑建议）
> **推荐开场钩子句：** "当事人来找我的时候说，花光六个钱包买的房，结果[问题]——我听了都觉得心疼。"

**结构：'他来找我的时候'→ 他的买房经历了什么 → 问题出在哪 → 我们怎么帮他 → 结果 → 替买房人说**

示例框架：
- **当事人出场**："当事人来找我的时候，[状态描述]。他花光积蓄买了[哪里的房]，结果[问题]。"
- **问题在哪**："我们一查发现，[争议焦点——产权/合同/违约/过户]出了问题。说实话，这种坑太常见了，但第一次遇到的人根本想不到。"
- **我们的策略**："我们帮他[诉讼/调解/协商]，核心争的是[法律依据]。"
- **结果**："最终[结果]。当事人说[他的反应]。"
- **替他说的话**："买房是普通人一辈子最大的交易之一。我想替他说的是——不是所有坑你都躲得掉，但至少有人能帮你爬出来。"
- **结尾**："你们买房遇到过什么问题？评论区说说。"

### 9. 奇葩案件/娱乐化案件
> **推荐钩子：** G情绪钩子（"离谱到无语"） / C反常识对比（层层反转）
> **推荐开场钩子句：** "我当事人来找我的时候，我以为是普通案子——结果越查我越懵，做了十年律师第一次见这种操作。" 

**定位**：专门处理"离谱""奇葩""让人震惊"的案件，娱乐化叙事，高传播性。
**适用**：奇葩离婚案、荒诞维权案、离奇纠纷、社会热点奇案。
**注意**：适合吃瓜型文案类型，不适合分析型。

**结构：**

```
爆点开场（最离谱的放第一句）
→ 人物引入（化名 + 标签，让观众一秒记住谁是谁）
→ 事件原委（正常叙事，铺垫）
→ 第一层反转（你以为这就是离谱的？）
→ 第二层反转（还有更炸裂的——）
```

## 叙事技巧

### 悬念式开场
- 用具体数字/金额开场："[X]万赔偿金，一分不少。"
- 用反转式提问："你以为[常识]？法院告诉你不是。"
- 用情感共鸣："如果是你的家人遇到这种事..."
- **吃瓜型开场**："今天这个案子我看了都摇头。" / "做了[年]律师，这种要求我第一次见。"

### 人物处理（V5.1 更新）
- **不使用任何人物名称或标签** — 统一用"当事人"/"他"/"她"
- 禁止任何形式的指代：❌ "老张""王姐""绿帽哥""洗头店老板娘安琪"
- 当事人出场统一："有个当事人来找我们" / "当事人来找我的时候"
- 用"你"拉近距离："你可能觉得这事离你很远，但其实..."

### 冲突强化
- 双方诉求对比："对方说...我们说..."
- 时间压力："距离诉讼时效只剩[X]天"
- 悬殊对比："一个大公司 vs 一个普通人"
- **吃瓜型冲突**：层层加码，每次反转比前一次更离谱

### 节奏控制
- 短句 + 长句交替
- 关键数字单独成句
- 设问引入："结果怎么样？"（停顿后回答）
- **吃瓜型节奏**：停顿前置——在爆点前留 `/` 或 `//`，让观众心理准备好

# 观点型模板

## 适用场景

从案件中能提炼出一个**独立于具体案情、对大众有普遍参考价值**的法律认知或反常识观点时使用。

## 推荐钩子
> **推荐钩子：** C反常识对比（反常识观点） / B顿悟发现（"做了律师才知道"） / A建议祈使（"建议你记住"）

## 结构

```
反常识观点开场（钩子句，跟具体案子无关）
· 为什么这个观点成立（法律逻辑/原理）
· 用一个案子佐证（"比如我们办过一个案子..."，一笔带过）
· 所以你应该怎么做（可操作建议）
```

## 撰写要点

| 要点 | 说明 | 正确 | 错误 |
|------|------|------|------|
| 钩子句 | 不能提到具体案子 | "离婚协议写暂不分割等于永远不分割？" | "夏先生离婚时在协议写了暂不分割..." |
| 案子角色 | 论据，不是故事 | "比如我们办过一个案子..." | 按时间线讲整个案子 |
| 观点展开 | 法律逻辑在前 | "法院看三个东西..." | 先讲案子再归纳 |
| 结尾 | 行动建议 | "签合同前记住三点" | "签了保证书不等于有保障..." |

# 分析型模板

## 案件类型与推荐分析框架

### 1. 通用分析框架（适用于大多数案件）

**结构：案情概览 -> 争议焦点 -> 法律分析 -> 判决逻辑 -> 实务启示**

### 2. 一审判决定性案件
**结构：原告诉求 -> 被告抗辩 -> 法院查明 -> 判决理由 -> 判决结果 -> 实务点评**

### 3. 二审/再审改判案件
**结构：一审结果 -> 上诉理由 -> 二审改判要点 -> 前后对比 -> 改判原因分析 -> 实务启示**

### 4. 多争议焦点案件
**结构：案件概要 -> 逐项分析（焦点1 + 焦点2 + 焦点3等）-> 综合评判 -> 实务启示**

## 四、完整输出示例

### 示例 1：分享型 — 交通事故案

```
你被车撞了，对方全责，保险公司只赔2万——你会接受吗？我当时就跟当事人说，这钱我们不能认。（分享型）

📋 核心观点：交通事故赔偿不止交强险，可以主张三期费用+精神损害 | 来源案件：机动车交通事故责任纠纷 | 人设风格：犀利分析型 | 总字数：260字 | 预估时长：58秒 | 适配平台：抖音
🎣 爆款钩子：C反常识对比（主）+ G情绪钩子（辅）

---

当事人来找我的时候，腿还拄着拐。他被一辆货车撞了——交警认定对方全责。对方保险公司上来就说：医疗费1.8万，再加上其他杂七杂八，一共给你2万，够意思了。

我一听就笑了——够意思？我告诉他，这个钱我们不能认。

交强险的医疗限额只有1.8万，但这只是起步。法律上的赔偿远不止这些——误工费、护理费、营养费、伤残赔偿金、精神损害抚慰金，每一项都可以单独主张。

我们帮他做了三期鉴定，把他的误工期、护理期、营养期全部固定下来，然后重新计算赔偿清单。

法院最终判了18.6万——我们跟他说过的话，一分不少。

所以说，交通事故不是对方给多少你就拿多少。赔偿清单列清楚再签字，别被保险公司几句话唬住。
```

### 示例 2：观点型 — 夫妻保证书案

```
你老婆写的保证书，法院认不认？我告诉你——很多时候，写了也白写。（观点型）

📋 核心观点：夫妻间的保证书不等于商业合同，法院不认 | 来源案件：离婚后财产纠纷 | 人设风格：犀利分析型 | 总字数：230字 | 预估时长：50秒 | 适配平台：视频号
🎣 爆款钩子：C反常识对比（主）+ A建议祈使（辅）

---

白纸黑字写了亏了全额赔偿，老婆签了字，老公拿着来要钱——法院怎么说？无效，不用赔。

很多人觉得写了保证书就能要到钱，但在法律上，夫妻间的保证不等于商业合同。法院会看三个东西：第一，这件事是不是夫妻共同决定的？第二，老婆写保证书的目的是什么——是为了鼓励老公拿钱出来，不是签商业担保合同。第三，有没有恶意——炒股亏了不是转移财产。

比如我们办过一个案子，老婆写了三份保证书，老公拿着来要17万。法院判了保证书无效，不用赔。

所以记住：夫妻间写保证书，想清楚再写。大事签正式合同，小事别拿保证书当法律武器。
```
