# Legal Script Kit — 律师口播文案创作工具

做律师短视频，最难的不是法律功底，而是**把复杂的案子变成观众听得懂、愿意听的话**。

这个工具就是帮您解决这个问题的。

## 一键安装

```bash
git clone git@github.com:lebiai/legal-script-kit.git
cp -r legal-script-kit ~/.codex/skills/
```

装完重启 Codex 即可使用。

## 它能做什么

| 您给它 | 它给您 |
|--------|--------|
| 一个判决书或案情摘要 | 可以直接对着镜头念的口播文案 |
| 一个法律观点 | 一篇有钩子、有案例、有态度的普法文案 |
| 一个奇葩案件 | 一篇能吃瓜、有反转、能出圈的爆款文案 |

## 内置方法论

- **5 种文案类型** — 分享型 / 分析型 / 吃瓜型 / 观点型 / 观点干货版
- **8 种爆款钩子** — 从「建议祈使」到「情绪钩子」，每种都有案例句
- **3 种人设风格** — 犀利毒舌 / 亲切邻家 / 幽默吐槽
