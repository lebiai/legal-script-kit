# 输出模板

```markdown
# [钩子句]（[文案类型]）

📋 核心观点：一句话总结 | 来源案件：[案由/—] | 人设风格：[风格] | 总字数：[N]字 | 预估时长：[N]秒 | 适配平台：[平台]
🎣 爆款钩子：[主钩子代码] + [辅钩子代码]（如 C + G）

---

## 口播文案

[文案正文，含句间表现标记]
```

## 元数据字段说明

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 核心观点 | ✓ | 一句话总结 |
| 来源案件 | ✓ | 案由或"—"（观点干货版用—） |
| 人设风格 | ✓ | 犀利分析型/亲切邻家型/幽默吐槽型 |
| 总字数 | ✓ | 汉字字数 |
| 预估时长 | ✓ | 字数÷语速 |
| 适配平台 | ✓ | 抖音/视频号/小红书 |
| 爆款钩子 | ✓ | 主+辅，如 C+G |

## 表现标记

| 标记 | 含义 |
|------|------|
| `[抬头看镜头]` | 直视镜头增强互动 |
| `[放慢]` | 降低语速强调关键信息 |
| `[停顿2秒]` | 留白给观众消化 |
| `[比划]` | 手势辅助说明 |
