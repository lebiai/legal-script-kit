# Step 4：润色 & Step 5：输出

## 润色检查清单

### ✓ 一致性
- 开头立场与结尾立场一致
- 全文第一人称统一用"我们/我"

### ✓ 去除废话
- 去掉"法律是公正的"、"法院是讲道理的"等
- 去掉"有法律问题找律师"等营销废话

### ✓ 顺滑度
- 段落之间必须有过渡句
- 每段不超过 3 句话

### ✓ 句间表现标记
位置：情感转折点、关键观点输出前、悬念揭示前
标记：`[抬头看镜头]` `[放慢]` `[停顿2秒]` `[比划]`
密度：全篇 3-5 处

### ✓ 平台适配

| 要素 | 抖音 | 视频号 | 小红书 |
|------|------|--------|--------|
| 标题风格 | 冲击力钩子 | 悬念式 | 评论区风 |
| 字幕要求 | 大字+强对比 | 适中 | 花字+表情 |
| 时长偏好 | 15-30秒 | 30-60秒 | 30-60秒 |

### ✓ 结尾设计
- 一定要给观众下一个看的理由
- 互动话术要具体

## 输出格式

```markdown
# [钩子句]（[文案类型]）

📋 核心观点：一句话总结 | 来源案件：[案由/—] | 人设风格：[风格] | 总字数：[N]字 | 预估时长：[N]秒 | 适配平台：[平台]
🎣 爆款钩子：[主钩子代码] + [辅钩子代码]（如 C + G）

---

## 口播文案

[文案正文，含句间表现标记]
```

## 字数控制

- 不少于 400 字
- 超过拆集（语速按 6字/秒）
- 语速参考：犀利分析型 4.5字/秒、亲切邻家型 3.5字/秒、幽默吐槽型 4字/秒
